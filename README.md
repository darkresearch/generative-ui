# Generative UI

A monorepo containing libraries for building generative AI user interfaces.

## Packages

- **[streamdown-rn](./packages/streamdown-rn)**: A React Native streaming markdown renderer with dynamic component injection.
- **[galerie-rn](./packages/galerie-rn)**: A React Native spatial canvas library for LLM-generated UIs.

## Development

This monorepo is managed with [Bun](https://bun.sh).

### Commands

- `bun install`: Install dependencies
- `bun run build`: Build all packages
- `bun run test`: Run tests for all packages
- `bun run type-check`: Type-check all packages
